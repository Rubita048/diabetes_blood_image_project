DATA DIRECTORY
----------------------
Bu klasör, proje için kullanılan özellik (feature) dosyalarını içerir.

Dosyalar:
- features_week1.csv → 1. hafta toplanan özellikler
- features_week2.csv → 2. hafta eklenen yeni veriler
- generate_sample_data.py → Test için örnek veri oluşturucu
- inspect_data.py → Veriyi hızlıca analiz etme aracı

Her CSV aşağıdaki sütunları içerir:
[glucose, insulin, bmi, age, blood_pressure, cholesterol, label]
